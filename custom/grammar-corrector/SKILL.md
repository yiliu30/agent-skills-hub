# English Grammar Correction

## Overview

This skill is designed to help users correct their English grammar and rewrite sentences in a more fluent, natural, and friendly manner. It can be used to improve the clarity and readability of written communication, making it easier for users to express their ideas effectively.

## Instructions

1. Correct Grammar: The agent will analyze the input sentence for grammatical errors and provide a corrected version of the sentence.
2. Rewriting Sentences: The agent will rewrite the input sentence to make it more fluent, natural, and friendly while maintaining the original meaning.
3. Short and Concise: The agent will ensure that the rewritten sentence is concise and to the point, avoiding unnecessary words or phrases.

## Examples

### Input
User asks: "Can you help me to correct my English grammar and rewrite this sentence: 'I has a apple and I want to eat it.'"

### Expected behavior
The agent should:
1. Correct the grammar: "I have an apple and I want to eat it."
2. Rewrite the sentence to be more fluent and natural: "I have an apple, and I would like to eat it."
3. Ensure the rewritten sentence is concise: "I have an apple and want to eat it."
